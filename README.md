# Elvesandjeger effect analysis workflow
The following workflow consists of 5 steps to:
1) Preprocessing of the data
2) Calculate the density of elvesandjeger per clearing site
3) Model the effect of the treatment on the elvesandjeger populations
4) Transform the data to long format
5) Publish the results as figures.

# Effektovervåking report
